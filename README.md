# GestureX ✋ - The Era of Contactless Interaction

AI-Based Gesture Recognition System.

GestureX is a computer vision project that allows users to interact with a computer using **hand gestures instead of traditional input devices like a mouse or keyboard**.

The system detects hand movements in real time using **MediaPipe and OpenCV** and converts gestures into meaningful actions.

---

## 🚀 Features

* Real-time hand gesture detection
* Contactless computer interaction
* AI-based hand tracking using MediaPipe
* User-friendly interface
* Modular architecture for easy expansion

---

## 🧠 Technologies Used

* Python
* OpenCV
* MediaPipe
* NumPy
* Computer Vision

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/mansiverma10457/GestureX.git
```

Navigate to the project folder:

```
cd GestureX
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the project:

```
python main.py
```
